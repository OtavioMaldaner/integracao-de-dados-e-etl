# Integração de Dados e ETL

Projeto desenvolvido para a disciplina de **Banco de Dados II** do curso de **Análise e Desenvolvimento de Sistemas**, focado na exploração de conceitos de Integração de Dados e ETL.

- O conteúdo teórico e a apresentação completa estão disponíveis no [PDF da Apresentação](./Apresentação%20-%20Integração%20de%20Dados%20e%20ETL.pdf).

## Atividade Prática

A atividade prática consiste em um processo de ETL desenvolvido na plataforma **[Databricks](https://login.databricks.com/signup?provider=DB&l=pt-BR)**.

- **Dataset:** Os dados utilizados são do [Brazilian Soccer Database](https://www.kaggle.com/datasets/ricardomattos05/jogos-do-campeonato-brasileiro/code), disponíveis no Kaggle.
- **Arquivos:** Os notebooks da atividade estão na pasta `/notebooks` e os arquivos de dados brutos estão na pasta `/data`.
